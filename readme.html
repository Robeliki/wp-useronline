From v1.0x To v2.05
  1. Deactivate WP-UserOnline Plugin
  2. Open Root WordPress Folder
  3. Delete file if exists:
    File: wp-useronline.php
  4. Open wp-content/plugins Folder
  5. Put:
    Folder: useronline
  6. Delete file if exists:
    File: useronline.php
  7. Open wp-content/plugins/useronline Folder
  8. Delete file if exists:
    File: wp-useronline.php
  9. Activate WP-UserOnline Plugin
  10. Refer To Usage For Further Instructions
» Usage Instructions
General Usage (Without Widget)
  1. Open wp-content/themes/<YOUR THEME NAME>/sidebar.php
  2. Add Anywhere:
    <?php if (function_exists('useronline')): ?>
       <li>
          <h2>UserOnline</h2>
          <ul>
             <li><div id="useronline-count"><?php get_useronline(); ?></div></li>
          </ul>
       </li>
    <?php endif; ?>
General Usage (With Widget)
  1. Activate WP-UserOnline Widget Plugin
  2. Go to 'WP-Admin -> Presentation -> Sidebar Widgets'
  3. Drag the UserOnline Widget to your sidebar
  4. You can configure the UserOnline Widget by clicking on the configure icon
  5. Click 'Save changes'
  6. Scroll down for instructions on how to create a UserOnline Page.
UserOnline Page
  1. Go to 'WP-Admin -> Write -> Write Page'
  2. Type any title you like in the post's title area
  3. Type '<page_useronline>' in the post's content area (without the quotes)
  4. Type 'useronline' in the post's slug area (without the quotes)
  5. Click 'Publish'
UserOnline Stats (Outside WP Loop)
  • To Display Most Number Of Users Online
  • Use:
    <?php if (function_exists('get_most_useronline')): ?>
       <p>Most Users Ever Online Is <?php get_most_useronline(); ?> On <?php get_most_useronline_date(); ?></p>
    <?php endif; ?>
  • To Display Users Browsing Site
  • Use:
    <?php if (function_exists('get_users_browsing_site')): ?>
       <div id="useronline-browsing-site"><?php get_users_browsing_site(); ?></div>
    <?php endif; ?>
  • To Display Users Browsing A Page
  • Use:
    <?php if (function_exists('get_users_browsing_page')): ?>
       <div id="useronline-browsing-page"><?php get_users_browsing_page(); ?></div>
    <?php endif; ?>
